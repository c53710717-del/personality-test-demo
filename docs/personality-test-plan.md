# 人格测试生成器 Demo 计划

## 1. 产品定位

这是一个“轻心理学 + 主题化表达”的测试生成网页，不做心理诊断，不输出医疗或治疗建议，定位为：

- 适合职场场景的沟通风格 / 协作风格测试生成器
- 支持创作者把抽象人格解释包装成更有传播性的主题体验
- 支持用户生成专属测试链接，发给二级用户完成答题和查看结果

建议前台统一使用这类表述：

- `人格风格`
- `协作倾向`
- `沟通偏好`
- `决策路径`

避免直接宣称：

- `专业诊断`
- `心理治疗结论`
- `精神健康评估`

## 2. 首版建议固定的心理流派

下面这 5 类最适合首版产品，既有心理学可信度，也容易做成大众愿意传播的测试。

### A. 积极心理学

适合原因：

- 强调优势、韧性、成就感与关系资源
- 比较适合职场、团队建设、自我探索
- 结果解释天然偏正向，分享阻力低

适合产出的人格风格：

- 优势激发型
- 韧性修复型
- 氛围点亮型
- 愿景召集型

### B. 认知行为取向（CBT）

适合原因：

- 关注想法、判断、行动之间的关系
- 很适合做“决策风格”“解决问题风格”“压力下反应模式”
- 问题设计清晰，容易结构化出题

适合产出的人格风格：

- 证据分析型
- 方案拆解型
- 风险预判型
- 执行闭环型

### C. 接纳承诺疗法（ACT）

适合原因：

- 关注价值观、心理灵活性、行动选择
- 很适合现代职场与自我成长叙事
- 结果报告可以做得很有“方向感”

适合产出的人格风格：

- 价值罗盘型
- 边界守护型
- 实验行动型
- 留白观察型

### D. 系统式视角（家庭系统 / 组织系统延展）

适合原因：

- 很适合团队协作、组织角色、沟通网络等主题
- 用户容易理解“我在关系系统里通常扮演什么角色”
- 特别适合 B 端或团队场景

适合产出的人格风格：

- 接口协调型
- 结构编织型
- 张力转译型
- 节奏稳定型

### E. 荣格原型 / 叙事心理学

适合原因：

- 天然适合做像 MBTI 一样有记忆点、有故事感的类型名
- 很适合和桌游、哲学、猫狗、世界观等方向结合
- 最容易做成“有梗、有传播性”的结果页

适合产出的人格风格：

- 探索者
- 守望者
- 召集者
- 炼金者

## 3. 首版建议固定的方向分类

为了同时覆盖职场传播性和趣味性，建议先固定 8 个方向：

### 核心职场类

- `职场协作`
- `创业团队`
- `关系沟通`

### 高传播兴趣类

- `桌游`
- `猫咪`
- `狗狗`

### 价值表达类

- `女性主义`
- `哲学流派`

### 扩展机制

- 保留 `自定义方向` 输入框
- 用户可输入任意关键词，比如：`二次元`、`咖啡馆`、`摇滚乐`、`剧本杀`

## 4. 首版推荐组合

如果我们要做一个最稳的 MVP，建议优先推这几组：

- `ACT × 职场协作`
- `积极心理学 × 创业团队`
- `CBT × 关系沟通`
- `荣格原型 × 桌游`
- `系统式视角 × 女性主义`
- `荣格原型 × 哲学流派`
- `ACT × 猫咪`
- `积极心理学 × 狗狗`

这样既有“职场可信度”，也有“社交传播感”。

## 5. Demo 版本的用户流程

### 创建者流程

1. 选择一个心理流派
2. 选择 1 到多个方向
3. 选择人格类型数量和题目数量
4. 系统生成：
   - 一套人格体系名称
   - N 个人格类型
   - 对应测试题
   - 一个可分享链接

### 二级用户流程

1. 打开分享链接
2. 完成测试题
3. 获得结果页：
   - 主人格类型
   - 次人格倾向
   - 职场/关系/行动特征说明
   - 与所选方向联动的主题化解释

## 6. Demo 范围

Demo 先不做后端数据库，直接用前端本地生成来验证：

- 生成器表单是否顺手
- 人格类型命名是否有吸引力
- 测试题是否像样
- 结果页是否足够有分享欲
- 分享链接方案是否成立

## 7. 正式版本建议拆分

### Phase 1：静态生成版

- 前端生成测试
- URL 编码分享
- 结果页本地计算

### Phase 2：可保存版

- 测试配置写入数据库
- 生成短链接
- 支持查看二级用户填写数量

### Phase 3：创作者增强版

- 支持编辑人格名称和解释文案
- 支持上传品牌封面图
- 支持导出海报 / 卡片
- 支持结果数据统计

## 8. 需要尽快确认的产品原则

- 结果是偏“娱乐传播”还是偏“职场工具”
- 用户创建测试时，是完全自由生成，还是先给模板库
- 结果页是否允许创建者二次编辑
- 是否要做“匿名作答”和“填写人数统计”

## 9. 当前建议

为了最快拿到一个像样的第一版，我建议：

- 先以 `轻职场 + 轻心理学 + 可传播主题包装` 为主线
- 首版固定 5 个心理流派和 8 个方向
- 结果页保持 `像 MBTI 一样有类型感，但不过度科学化`
- 用 Demo 先验证类型命名、解释风格、分享意愿
